To change setting to run the code for another transformation program:
	1. In setting.java line 23-46, 4 parameters for every transformation programs is initialized
	(firstecorename,secondecorename,sourcemetamodel,targetmetamodel), if we put comment for 3 transformations
	and uncomment the one we plan to run.
	2. delete config file which is located in cd.udem.fixingatlerror/examples/config.text and copy 
	 related config file for the transformation program which is located in CorrectOutputModel folder
	3. delete all files in semantic faulty folder which is located in cd.udem.fixingatlerror/examples/class2rel/trafo/semanticfaulty
	  and select faulty transformation files from related transformations from SemanticFaultyTransformations folder.
	4. delete 4 files (correctoutput0.xmi, correctoutput1.xmi, correctoutput2.xmi, correctoutput3.xmi) which
	  are located in examples and select related 4 correctoutput.xmi files from CorrectOutputModels folder.
